Dynamic memory allocation - 10.9
Control allocation and deallocation of memory in a program 
the heap is a region of memory assigned to each program for storing objects dynamically allocated at execution time
The new operator allocates storage of the proper size for an object
To destroy a dynamically allocated object use the delete operator
